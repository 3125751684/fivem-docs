事件名称
----------
```
baseevents:enteredVehicle
```

参数
----------

```
vehicle currentVehicle, int currentSeat, string vehicleDisplayName
```

- **currentVehicle**: 玩家刚进入载具的句柄
- **currentSeat**: 玩家坐在车里的座位号（-1是驾驶员座位，0代表乘客右前坐等）。
- **vehicleDisplayName**: 一个字符串，其中包含玩家刚刚输入的车辆的显示名称。
