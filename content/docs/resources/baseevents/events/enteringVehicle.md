事件名称
----------
```
baseevents:enteringVehicle
```

参数
----------

```
vehicle targetVehicle, int vehicleSeat, string vehicleDisplayName
```

- **targetVehicle**: 玩家试图进入的载具句柄
- **vehicleSeat**: 玩家试图进入的座位号（-1是驾驶员座位，0代表乘客右前坐等）。
- **vehicleDisplayName**: 一个字符串，其中包含玩家试图进入的车辆的显示名称。
